# Code of Conduct

## Our Commitment

Team Zero Downtime is committed to maintaining a respectful, professional, and collaborative environment.

## Expected Behaviour

- Be respectful.
- Be constructive.
- Communicate professionally.
- Welcome feedback.
- Share knowledge.

## Unacceptable Behaviour

- Harassment
- Discrimination
- Personal attacks
- Offensive language
- Deliberate disruption

## Decision Making

Technical decisions should be based on:

- Business requirements
- Security
- Maintainability
- Performance
- Simplicity

## Enforcement

Project maintainers reserve the right to remove contributions that violate this Code of Conduct.
