# Security Policy

## Reporting Security Issues

Do not publicly disclose security vulnerabilities.

Instead, report them privately to the project maintainers.

## Scope

Security reports may include:

- Authentication issues
- Authorization bypass
- Payment vulnerabilities
- Webhook validation issues
- Organization isolation issues
- Sensitive data exposure

## Security Principles

UnityFund follows these principles:

- Least privilege
- Secure by default
- Defense in depth
- Auditability
- Financial integrity

## Responsible Disclosure

Valid security reports will be investigated promptly.

Security fixes should be released before public disclosure whenever possible.
